# Polyglot Actor Orchestrator

A modular, multi-LLM chat interface built with Next.js, Vercel Blob, and the "Master Router" architecture.

## ðŸ›  Setup Instructions

### 1. Vercel Environment Variables
Add the following to your Vercel Project Settings:
- `OPENAI_API_KEY`: Your OpenAI secret key.
- `GEMINI_API_KEY`: Your Google AI Studio key.
- `BLOB_READ_WRITE_TOKEN`: Automatically generated by Vercel Blob.
- `MASTER_ROUTER_MODEL`: e.g., `gpt-4o-mini`

### 2. Vercel Blob Configuration
Ensure you have initialized a **Blob Store** in the Vercel "Storage" tab. The application expects a structure of `actors/*.json` and `chats/*.json`.
